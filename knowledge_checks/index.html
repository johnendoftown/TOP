How do you create a paragraph in HTML?>

A paragraph is created using the "p" tag.

How do you create a heading in HTML, how many levels of headings are there, and what is the difference between each one?

A heading is created with the "h" tag, followed by a number from 1 to 6, like "h1". In this case, all of the headings in this document are level 1 headers, they get smaller as the number increases.

What element should be used to make text bold and/or italic?

The "strong" and "em" tags can be used to make text bold and italic respectively. These are used so that accessability technology can properly address bold and italic text on a website.

What relationship does an element have with any nested elements within it?

The element with nested elements within it is the parent.

What relationship do two elements have if they are at the same level of nesting?

They are siblings.
