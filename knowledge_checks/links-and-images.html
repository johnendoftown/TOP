Knowledge Check: Links and Images

What element is used to create a link?

The "a" element is used to create a link

What is an attribute?

An attribute is an additional parameter which holds a value relevant to the tag

What attribute tells links where to go to?

The "href" attribute tells links where to go

What is the difference between an absolute and relative link?

An absolute link links to an external website or url, whereas a relative link is an internal link inside the website

Which element is used to display an image?

The element used to display an image is the "img" element

What two attributes do images always need to have?

An image always needs "src" and "alt" attributes. additionally you can ise "width" to edit image size

How do you access a parent directory in a filepath?

To accesss a parent directory, use "../" before the relative link to go up a directory

What are the four main image formats that you can use for images on the web?

  • .jpg, a smaller sized image
  • .png, a larger and prettier image
  • .gif, an animated picture
  • .src, a scalable image
